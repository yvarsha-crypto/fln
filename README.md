**Version 2.0** | Classes 1–3 | NCERT · CBSE · ICSE · State Boards

## What is this?

This repository contains the complete **Foundational Literacy & Numeracy (FLN)** Mathematics level structure for Classes 1–3, aligned with:
- NCERT National Curriculum Framework (NCF 2023)
- CBSE syllabus (including NEP 2020 revisions)
- ICSE syllabus
- All major Indian state boards (AP, Telangana, Maharashtra, Karnataka, TN, Rajasthan, UP, Gujarat, MP, Odisha)
- NIPUN Bharat competency benchmarks

## Sub-Level System

Each level has 3 difficulty tiers:
- **X.0** — Main level (standard NIPUN benchmark difficulty)
- **X.1** — Easier variant (for students who need more practice)
- **X.2** — Remediation (most scaffolded, for struggling students)

Rule: A student must clear X.0 before progressing. If they fail, route to X.1 → X.2 → re-attempt X.0.

## NIPUN Bharat Alignment

These levels map directly to NIPUN Bharat's foundational numeracy goals:
- **Goal 1**: Number sense (Levels 1–13)
- **Goal 2**: Arithmetic operations (Levels 7–8, 16–17, 27, 34–36)
- **Goal 3**: Measurement & geometry (Levels 24–25, 32, 38–39)
- **Goal 4**: Patterns & data (Levels 9, 40)

Expected achievement: All students should reach at least Level 23 by end of Class 2, and Level 40 by end of Class 3.
