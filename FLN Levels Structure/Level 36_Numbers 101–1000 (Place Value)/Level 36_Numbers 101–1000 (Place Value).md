# Level 36: Numbers 101–1000 (Place Value)

**Class:** Class 3 | **Age Group:** Age 8-9

## **Objective**

Develop an understanding of three-digit numbers by identifying hundreds, tens, and ones, recognizing number patterns, and completing number sequences up to 1000\.

## **Description**

Students learn to recognize, read, write, and represent numbers from 101 to 1000\. 

They identify the hundreds, tens, and ones in three-digit numbers, complete number sequences, identify missing numbers, and build numbers using place value. 

Activities use both numerical and visual representations to strengthen number sense.

## **Learning Outcome**

Students can:

* Recognize numbers from 101–1000.  
* Identify hundreds, tens, and ones.  
* Represent numbers using place value.  
* Complete number sequences.  
* Identify missing numbers.  
* Build three-digit numbers using place value.

## **Topics Covered**

* Numbers 101–1000  
* Hundreds  
* Tens  
* Ones  
* Place Value Chart  
* Missing Numbers  
* Number Sequences  
* Number Formation

## **Example Activities**

* Identify hundreds, tens, and ones.  
* Complete place value charts.  
* Fill missing numbers.  
* Complete number sequences.  
* Build numbers from place values.
