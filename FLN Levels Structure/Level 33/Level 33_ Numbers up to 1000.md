# Level 33: Numbers up to 1000

## Concept
Reading, writing, and understanding place value (hundreds, tens, ones) for numbers up to 1000.

## Objective
Read, write, compare, and order 3-digit numbers; understand expanded form.

## Description
Students extend place value to the hundreds column. They read and write 3-digit numbers, represent them in expanded form (e.g., 347 = 300 + 40 + 7), and compare/order them using >, <, =.

## Learning Outcome
- Read and write numbers up to 1000.
- Identify hundreds, tens, and ones in 3-digit numbers.
- Write expanded form and compare 3-digit numbers.
