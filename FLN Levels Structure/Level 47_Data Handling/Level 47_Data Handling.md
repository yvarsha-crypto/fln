# **Level 47: Data Handling** 

**Class:** Class 3 | **Age Group:** Age 8-9

## **Objective**

Develop the ability to collect, represent, interpret, and compare data using tally marks, pictographs, and simple bar graphs.

## **Description**

Students strengthen their data handling skills by organizing information using tally marks, interpreting pictographs, and reading simple bar graphs. 

They compare quantities, identify the greatest and smallest values, and answer simple questions based on visual data representations.

## **Learning Outcome**

Students can:

* Read and interpret tally charts.  
* Read pictographs using a given key.  
* Read simple bar graphs.  
* Compare quantities using data representations.  
* Identify the greatest, smallest, and equal values.  
* Answer simple questions based on data.

## **Topics Covered**

* Tally Marks  
* Frequency Tables  
* Pictographs  
* Bar Graphs  
* Data Comparison  
* Data Interpretation
