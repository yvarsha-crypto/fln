# Level 37: Fractions — Equivalent and Simple Operations

## Concept
Identifying unit fractions, equivalent fractions, and adding/subtracting like fractions.

## Objective
Understand unit fractions, find equivalent fractions, and add/subtract fractions with the same denominator.

## Description
Building on Level 30 (halves and quarters), students now understand fractions as parts of a whole with different denominators (1/3, 1/4, 1/5, 2/3, 3/4). They identify equivalent fractions pictorially and add/subtract fractions with the same denominator.

## Learning Outcome
- Identify and represent unit fractions up to 1/10.
- Recognise equivalent fractions using diagrams.
- Add and subtract fractions with the same denominator.
