# **Level 54: Fraction Operations**

**Class:** Class 4 | **Age Group:** Age 9-10

## **Objective**

Develop the ability to perform addition and subtraction of fractions with the same denominator using visual models and symbolic representations.

**Description**

Students extend their understanding of fractions by performing addition and subtraction of like fractions. 

Activities use visual fraction models, shaded figures, and everyday objects to strengthen conceptual understanding and computational fluency.

## **Learning Outcomes**

Students can:

* Add fractions with the same denominator.  
* Subtract fractions with the same denominator.  
* Represent fraction operations using visual models.  
* Identify the resulting fraction after combining or removing equal parts.  
* Apply fraction operations using picture-based activities.


## **Topics Covered**

* Addition of Like Fractions  
* Subtraction of Like Fractions  
* Visual Fraction Models  
* Fraction Representation  
* Picture-Based Fraction Applications
