# **Level 52: Maps & Directions**

**Class:** Class 4 | **Age Group:** Age 9-10

## **Objective**

Develop spatial reasoning and navigation skills by interpreting simple maps, identifying directions, locating positions, and following routes using visual representations.

## **Description**

Students learn to interpret simple maps using directions and landmarks. 

They identify positions, follow routes, determine locations using cardinal directions (North, South, East, and West), and apply spatial reasoning to solve navigation-based activities. 

The level strengthens observation, logical thinking, and real-life navigation skills.

## **Learning Outcomes**

Students can:

* Identify North, South, East, and West.  
* Read and interpret simple maps.  
* Locate places using directions.  
* Follow routes on a map.  
* Identify landmarks.  
* Determine the shortest or correct path between locations.  
* Apply map-reading skills in everyday situations.

## **Topics Covered**

* North, South, East, West  
* Position  
* Landmarks  
* Route Following  
* Map Reading  
* Shortest Path  
* Spatial Reasoning
