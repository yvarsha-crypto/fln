# Level 40: Data Handling

## Concept
Collecting, organising, and interpreting data using tally marks, tables, and picture graphs (pictographs).

## Objective
Collect data, represent it in a tally/table/pictograph, and answer questions from the graph.

## Description
Students collect simple data (favourite fruit, number of students in each house), record using tally marks, make frequency tables, and create/read pictographs. They answer questions: "Which is most/least common? How many more?"

## Learning Outcome
- Record data using tally marks.
- Organise data in a frequency table.
- Read and interpret a pictograph.
- Answer comparison questions from data.
