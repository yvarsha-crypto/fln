# **Level 50: Advanced Multiplication**

**Class:** Class 4 | **Age Group:** Age 9-10

## **Objective**

Develop proficiency in multiplying larger numbers by applying place value concepts and the standard multiplication algorithm to solve two-digit and three-digit multiplication problems.

## **Description**

Students extend their multiplication skills by solving multiplication problems involving two-digit and three-digit numbers. 

They apply multiplication strategies, place value understanding, and the standard algorithm to calculate products accurately and efficiently.

## **Learning Outcomes**

Students can:

* Multiply two-digit numbers by two-digit numbers.  
* Multiply three-digit numbers by one-digit numbers.  
* Apply the standard multiplication algorithm.  
* Use place value correctly during multiplication.  
* Solve real-life multiplication problems.

## **Topics Covered**

* Two-Digit × Two-Digit Multiplication  
* Three-Digit × One-Digit Multiplication  
* Vertical Multiplication  
* Place Value in Multiplication
