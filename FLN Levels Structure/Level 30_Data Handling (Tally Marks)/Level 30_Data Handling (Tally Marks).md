# Level 30: Data Handling (Tally Marks)

**Class:** Class 2 | **Age Group:** Age 7-8

### **Objective**

### Develop the ability to represent, interpret, and compare data using tally marks and visual data representations.

### **Description**

### Students learn to organize information using tally marks, read tally charts, and compare quantities represented through visual data. 

### Activities emphasize observation, grouping, and interpretation rather than simple counting.

### **Learning Outcome**

### Students can:

* ### Represent quantities using tally marks.

* ### Read and interpret tally marks.

* ### Compare quantities using tally charts.

* ### Identify the greatest and smallest quantities

### **Topics Covered**

* ### Counting Objects

* ### Tally Marks

* ### Reading Tallies

* ### Comparing Quantities

* ### Simple Data Representation

### **Example Activities**

* ### Count fruits and draw tally marks.

* ### Match tally marks to quantities.

* ### Identify which group has the most objects.

* ### Complete missing tally marks.
