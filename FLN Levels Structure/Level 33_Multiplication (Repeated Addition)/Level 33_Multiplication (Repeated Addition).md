# Level 33: Multiplication (Repeated Addition)

**Class:** Class 2 | **Age Group:** Age 7-8

### **Objective**

Develop an understanding of multiplication as repeated addition and equal grouping using the 2, 3, and 4 multiplication tables.

### **Description**

Students learn multiplication through visual groups, repeated addition, and number patterns. 

They understand that multiplication is a faster way of adding equal quantities repeatedly and apply this understanding using the 2, 3, and 4 tables.

### **Learning Outcome**

Students can:

* Identify equal groups.  
* Represent multiplication as repeated addition.  
* Write multiplication sentences.  
* Use the 2, 3, and 4 multiplication tables.  
* Solve simple multiplication problems.

### **Topics Covered**

* Equal Groups  
* Repeated Addition  
* Multiplication Sentences  
* Table of 2  
* Table of 3  
* Table of 4

### **Example Activities**

* Count objects arranged in equal groups.  
* Convert repeated addition into multiplication.  
* Complete multiplication facts.  
* Match groups to multiplication expressions.
