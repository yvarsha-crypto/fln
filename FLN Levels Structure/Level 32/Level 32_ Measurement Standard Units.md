# Level 32: Measurement — Standard Units

## Concept
Measuring length (cm/m), weight (kg/g), and capacity (litre/ml) using standard units.

## Objective
Measure and estimate length, weight, and capacity using standard units; choose appropriate units.

## Description
Building on Level 25 (comparison), students now use rulers to measure in cm, estimate weights in kg/g, and measure capacity in litres. They compare measurements and solve simple measurement problems.

## Learning Outcome
- Measure length using cm and m.
- Understand kg and g for weight; l and ml for capacity.
- Choose the appropriate unit for a given object.
