# Level 28: Multiplication — Introduction

## Concept
Multiplication as repeated addition; skip counting as the foundation for multiplication.

## Objective
Understand multiplication as repeated addition and use skip counting to find products.

## Description
Students connect skip counting (already learned in Level 20) to repeated addition and then to multiplication. Activities use equal groups of objects, number lines, and arrays. Tables of 2, 3, 4, and 5 are introduced.

## Learning Outcome
- Write repeated addition as multiplication.
- Use skip counting to find products for tables of 2, 3, 4, 5.
- Solve simple multiplication problems using objects or number lines.
