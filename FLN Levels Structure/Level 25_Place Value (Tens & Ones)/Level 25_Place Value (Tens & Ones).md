# Level 25: Place Value (Tens & Ones)

**Class:** Class 2 | **Age Group:** Age 7-8

## **Objective**

Develop an understanding of place value by identifying, representing, and composing two-digit numbers using tens and ones.

## **Description**

Students learn that every two-digit number is made up of tens and ones. 

Through place value charts, grouping activities, visual representations, and number-building exercises, they understand how digits represent different values depending on their position. 

This foundational understanding prepares students for number recognition up to 100, comparison, ordering, addition with carry, and subtraction with borrowing.

## **Learning Outcome**

Students can:

* Identify the tens and ones in a two-digit number.  
* Represent numbers using tens and ones.  
* Build two-digit numbers from given tens and ones.  
* Complete place value charts accurately.  
* Determine the number represented by groups of tens and ones.  
* Apply place value understanding to compare and order numbers.  
* Recognize that changing the position of digits changes the value of the number.

## **Topics Covered**

* Tens  
* Ones  
* Place Value  
* Place Value Chart  
* Number Formation  
* Representing Numbers  
* Building Numbers  
* Expanded Form (basic)  
* Grouping into Tens and Ones


  ## **Example Activities**

* Write the tens and ones for a given number.  
* Form a number using the given tens and ones.  
* Complete a place value chart.  
* Match numbers with their place value representation.  
* Count groups of tens and ones and write the number.  
* Identify missing tens or ones.  
* Connect visual representations with numbers.
