# FLN Levels Structure — Complete Data Reference

**Total Levels:** 59  
**Curriculum Scope:** Preschool 1 (Age 3-4) through Class 4 (Age 9-10)  
**Strands:** Number Sense, Number Operations, Shapes, Measurement, Patterns, Money, Calendar & Time, Fractions, Data Handling

---

## Level 1: Quantity Comparison

**Class:** Preschool 1 | **Age Group:** Age 3-4

### Main Overview
**Learning Outcome:** By the end of this level, the child will be able to:
- Recognize equal quantities
- Identify groups with more or less objects
- Compare quantities visually
- Develop observation and comparison skills
- Build a foundation for future mathematical operations

**Topics Covered:** Comparing Quantities, Equal, More and Less, Visual Comparison

**Skills Developed:** Visual discrimination, Quantity recognition, Logical thinking, Comparison skills, Understanding equal/more/less, Quantity awareness, Attention to detail, Decision making, Pattern observation

### Sub-Level 1.0 (Main Level)
- **Section A - Equal Quantity:** Contains 2 columns; child matches equal things
- **Section B - More/Less:** Questions in pairs; child circles the one with greater value

### Sub-Level 1.1 (Easier)
- Less difficult than main level
- Same structure as 1.0 but simplified

### Sub-Level 1.2 (Easiest)
- Least difficult; same structure with simplest content

## Level 2: Odd One Out

**Class:** Preschool 1 | **Age Group:** Age 3-4

### Main Overview
**Concept:** Odd One Out (Classification and Differentiation)

**Objective:** Identify the item that does not belong to the group

**Learning Outcome:** Identify similarities and differences among objects, Classify objects based on common characteristics, Recognize an item that does not belong, Develop logical reasoning and observation skills, Make decisions based on visual information

**Topics Covered:** Odd One Out, Classification and Differentiation, Similarities and Differences, Attribute-Based Grouping

**Skills Developed:** Classification, Observation, Logical reasoning, Critical thinking, Visual discrimination, Pattern recognition, Decision making, Problem solving, Categorization

### Sub-Level 2.0 (Main Level)
- Child selects the odd one out from a group of objects

### Sub-Level 2.1 (Easier)
- Easier than main level

### Sub-Level 2.2 (Easiest)
- Questions where child can clearly observe differences
- Easiest questions

## Level 3: Matching + Tracing Lines

**Class:** Preschool 1 | **Age Group:** Age 3-4

### Main Overview
**Section A - Matching:** Children develop matching and association skills by identifying relationships between objects, shapes, quantities, and concepts

**Topics Covered:** Matching and Association, Shape Recognition, Opposite Concepts, Object Relationships

**Learning Outcome:** Identify relationships between objects, Match similar items correctly, Associate quantities with corresponding groups, Recognize shapes and real-world representations, Understand simple opposite concepts

**Section B - Tracing Lines:** Develop fine motor control

### Sub-Level 3.0 (Main Level)
- **Section A:** Opposite Matching, Shape Matching, Relation Matching, Quantity Matching
- **Section B:** Horizontal lines, Vertical lines, Slanting lines, Semi Circle, Curve; contains only 2 dots (no trace lines provided, only reference dots)

### Sub-Level 3.1 (Easier)
- **Section A:** Same concepts, only 3 options per question
- **Section B:** Contains trace lines

### Sub-Level 3.2 (Easiest)
- **Section A:** Similar concepts, 2 columns to match
- **Section B:** Tracing activities

### Sub-Level 25.1 (Easier)

### Sub-Level 25.2 (Easiest)

- **Section C:** Place Value Chart (fill missing value)
